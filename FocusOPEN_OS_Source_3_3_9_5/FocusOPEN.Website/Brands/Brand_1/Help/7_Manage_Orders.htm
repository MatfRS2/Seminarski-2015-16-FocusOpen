Processing Orders

Available to: All Admin & Upload Users
[top ^ | close]

Once uploaded, catalogued and published on the system, assets are available for other users to browse and request for use. Assets for which no download rights have been specified can be downloaded by users without approval. However, users requesting assets for which download rights have been specified, must apply for usage approval before these assets can be downloaded from the system.

Users who want to use a restricted asset must first specify how they intend to use the asset. Having supplied information on intended usage, users will submit their order. The system will then dispatch an order approval notification to the asset contact (the user associated with the asset as specified when cataloguing).

To Respond To An Order

  • Click on the link provided in the notification email. The order approval screen is displayed. All assets which require usage approval ordered by a user are displayed.
  • Follow the steps below to approve, reject or postpone each request to use an asset in the order.

Finding Orders: View Order List

Available to: All Admin & Upload Users
[top ^ | close]

Alternatively, all orders (pending and completed) can be viewed via the View Orders List screen. To find an order:

  • Select 'Manage Orders > View Order List' from the main navigation in the Admin Area. The View Order List screen will be displayed.
  • The View Order List displays all orders made for assets uploaded, catalogued and published to the system by you. All orders are displayed by default (regardless of order status: pending or complete).
  • Order Ref#: To search for an order, enter the order reference number into the text field available. Note: You can also enter a user email address into this field to view all orders made by that user on the system.
  • Applicant: shows a list of all users with pending orders.
  • Press the 'Search' button to filter the results displayed in the table below. The View Orders List table displays the following column headings: Order Reference Number, Number of Assets in Order, Applicant Name, Brand, Order Status (Pending/Complete), Date Requested, Request Deadline (Date Required By).
  • To view orders pending only, check the 'show pending orders only' checkbox beneath the search controls. To sort results by a column heading, click on the link beneath the column title. Note: Applicants can be contacted via email by clicking on the link under the applicant's name in the Applicant Name column.

Usage Approval & Rejection

Available to: All Admin & Upload Users
[top ^ | close]

Having found an order pending, you can view the details of that order by clicking on the link under the order reference number. The Order Approval screen is displayed

To approval or reject the usage applications for each asset in an order:

  • Select either the 'Approve' , 'Reject' or 'Still Pending' radio button to decide whether or not an asset can be used as requested. If the 'Still Pending' radio button is selected, your decision is deferred.

    Note: The 'Your Response' text field can be used to provide the applicant with further information on your decision. For example, the reasons why an application has been rejected or further stipulations on approved asset usage. To contact a user for further information regarding their order, click on the link under the applicants name in the 'Placed By' field, to send an email to that user.

  • Once you have completed all assets in an order, press the 'Process Order' button to process the order. The applicant will be notified of your decisions via email. Note: The order status will remaining as 'pending' if any asset in the order retains the 'still pending' option.
