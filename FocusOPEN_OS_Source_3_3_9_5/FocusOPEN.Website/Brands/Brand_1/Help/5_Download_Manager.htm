Downloading Assets from the System: About the Download Manager

[top ^ | close]

Having placed an order with the system, approved assets and assets which do not require approval can be downloaded in the Download Manager, which is available from every interface in FocusOPEN Digital Asset Manager via the "Download Manager" link on the main menu. The Download Manager allows users to view orders made, view the status of assets in each order and download assets in different resolutions or batched in zipped files.

Viewing Orders in the Download Manager

[top ^ | close]

The Download Manager displays the content of a selected order below in the asset list. By default the latest order is shown. You can choose the view the last ten orders made or all orders you have made in FocusOPEN Digital Asset Manager. The reference number and date of the order being viewed are displayed in the "Currently Viewing Order" field.

View Last 10 Orders

The "View Last 10 Orders" dropdown menu is displayed by default and allows you to select one of the last ten orders made via the system to view. This feature can be toggled with the "View All Orders" list menu by selecting the "View All Orders" link to the right of the dropdown.

View All Orders

The "View All Orders" list menu allows you to select an order to view from all orders made via the system. This feature can be toggled with the "View Last 10 Orders" list menu by selecting the "View Last 10 Orders" link to the right of the list menu.

About Asset Status: Approved, Rejected, Pending & Expired

[top ^ | close]

Having selected an order to view, the contents of that order are displayed below in the asset list. The feedback to the far right of the asset thumbnail and details indicates the current status of each asset: approved (or approval not required), rejected, pending and expired. Only approved assets may be downloaded.

Approved & Approval Rejected

Only approved assets can be downloaded from the Download Manager. Assets either: do not require approval and may be downloaded immediately from the Download Manager, or: are approved for use on a case-by-case basis by an Upload User. On approval, these assets then become available to download via the Download Manager. If an Upload User rejects your application to use an asset, the asset will display an "approval rejected" notification to the right of the asset thumbnail and details in the Download Manager and will not be available to download. Further details regarding usage or why an application has been rejected can be viewed by clicking the "Show Approval Notes" beneath the asset details.

Note: Approval Notes are only available for assets which have required approval by an Upload User.

Approval Pending

If an asset in the asset list requires approval by an Upload User, but has not yet been administered, the asset status displays as "Approval Pending" to the far right of the asset thumbnail and details. This asset is not downloadable until approved. Note: You will receive an email notification to your registered inbox when a decision has been made.

Expired

To ensure that assets up-to-date and relevant to the mission and business, all assets are set to expire for reconsideration by System Administrators. Assets which have expired will continue to be displayed in the asset list, but are no longer available for download via the system. Expired assets display the status "Expired" to the far right of the asset thumbnail and details.

Downloading Assets in an Order

[top ^ | close]

Only approved assets can be downloaded from the Download Manager. When an asset is available to download, the "Download" button will be displayed beneath the asset details. Press the "Download" button to download the file in its original format to your local or corporate network drive. Note: Assets with large file sizes or batched zip files may take some time to download.

Setting the Download Resolution (Images Only)

The resolution of an image asset can be selected prior to download via the "Download Resolution" dropdown menu displayed beneath the asset details. Resolutions available are: 72dpi (for screen or web), 300dpi (for print) and 600 dpi (for print). Note: The higher the resolution, the larger the filesize. Assets with large filesizes may take longer to download. If an asset has a filesize in access of 10MB, a warning will be displayed to the right of the filesize.

Downloading Selected in a Zipped File

Assets can also be downloaded in batches in zip file format, so more than one asset can be downloaded at one time. To select an download a number of assets in an order:

  • Select the assets to download by checking the "Select This Asset" checkbox below the "Download" button. (Note: Only approved assets will display the "Select This Asset" checkbox.
  • Click the "Download As Zip" link from the top or bottom of the asset list.
  • Save the zip file to your local or corporate network drive.

Note: Check the "Select All Assets Listed" checkbox to select all approved assets in an order to download. Click the "Deselect All Assets" link to deselect all assets selected.
