Manage Homepage

Available to: Super Admin Only
[top ^ | close]

The Manage Homepage feature allows Super Administrators to upload new images and select searches or public lightboxes for users to view on the homepage. It is usually easier to carry out home page management with two browser windows open, one with the catalogue area visible, the other with the home page manager.

  • Copy on the left hand side of the page can be amended using the built in HTML editor.
  • Up to 4 image 'tiles' can be shown on the home page. To upload one, press the Browse button to select a file from your local or corporate network drive. Note :Images will be automatically resized to match required dimensions. Images must be uploaded in GIF or JPG format.
  • To link to a search, go to the search facility in the main catalogue, prepare your search with all required categories, filters and keywords. In the bottom right corner is an option 'link to search' right click (control-click on a mac) and copy the link or shortcut. Return to the home page area and paste the URL under the image.
  • To link to a lightbox, go to the lightbox are in the catalogue, find the required lightbox, there will be a link 'link to this lightbox'. Right click (control click on a mac) and copy the URL to the lightbox. Paste this into the URL field to link the image to the lightbox.
  • Any URL may be included under one of the image tiles, not just links to searches or URLs.
  • To preview changes before they are published, press the Preview button. A version of the homepage will be launched in a new browser window with your changes in situ. Note: These changes will not saved.
  • To publish changes live, press the Publish button. Note: These changes will be saved and will be made visible to all system users.

Manage Asset Types

Available to: Super Admin Only
[top ^ | close]

The Manage Asset Types feature allows Super Administrators to add, edit and delete asset or media type groups to avoid the need for separate filtering options for every kind of file type. The Asset Type manager works in combination with the File Extension Manager.

  • Select 'Manage Content > Manage Asset Types' from the main navigation in the Admin Area. The Manage Asset Types screen will be displayed.
  • To add a new type, enter the name in the field at the bottom of the page.
  • Click [add new] to assign file extensions to the asset type. This will take you to the File Extension Manager.
  • File extensions can only exist in a single asset type, it is not possible to assign an extension to multiple types simultaneously.
  • Asset Types cannot be deleted if they have assets assigned to them, however, they can be made invisible which will prevent them from being selected in searches and during cataloguing.

Manage File Extensions

Available to: Super Admin Only
[top ^ | close]

The Manage Extensions feature allows Super Administrators to add, edit and delete file extensions. This feature works in tandem with Manage Asset Types.

  • Select 'Manage Content > Manage File Extensions' from the main navigation in the Admin Area. The Manage File Extensions screen will be displayed.
  • The list of file extensions added so far will be shown. The drop down menu enables extensions to be filtered by asset type. The APS column indicates whether an asset will be processed by the Asset Processing Server - this generates a preview and thumbnail for the asset automatically so the user does not need to manually add one.
  • Click add new extension. You must enter the extension (the letters after the fimnal full stop in a file), the name of the file type and choose an asset type. You may also upload an optional icon which will be shown when viewing assets in a list or in the asset detail pop-up window. The asset type can also be made invisible.
  • Click save to add the file extension, if you have many similar extensions, you can also use save and duplicate to use the last saved extension.

Manage Categories

Available to: Super Admin Only
[top ^ | close]

The Manage Categories feature allows Super Administrators to add, edit, delete categories that assets are assigned to. This feature changes the category taxonomy search which is visible to users in the catalogue.

  • Select 'Manage Content > Manage Metadata > Manage Categories' from the main navigation in the Admin Area. The Manage Categories screen will be displayed.
  • The category taxonony will be shown down the left hand side. Click the arrow to the left of each folder to open a branch of the taxonomy.
  • To change an existing taxonony, click on the folder and it will populate the editing area to the right. The category name is the name that appears on the category link. External reference is an optional identifier for a third party classification scheme (if required). Category Message is text that appears to user when the category is opened in the catalogue. Synonyms are alternative terms for categories which match on searches but do not get shown to users.
  • To add a category, you must pick an existing one that it will become a child of. If this is a top level category, choose assets. Then enter the required details for the category and click update.
  • To change the position of a category in the taxonomy, use the direction controls (left, right, up, down) to move it or you can drag and drop the category into a different sub-folder.
  • Categories can only be deleted if they do not contain child categories or have assets assigned to them. The view asset list function on the asset menu enables assets in a given category to be moved or filtered (and then deleted.

Manage Metadata Values

Available to: Super Admin Only
[top ^ | close]

The Manage Metadata Values feature allows Super Administrators to add, edit and delete indexed metadata which is chosen from drop-down menus when cataloguing assets or searching. This metadata is most often used in the subject metadata section of the asset catalogue process. The operation is a two stage process where the user first selects the metadata they wish to work with and then changes it. Up to 12 different indexed metadata taxonomies can be defined.

  • Select 'Manage Content > Manage Metadata > Manage Metadata Values' from the main navigation in the Admin Area. The Manage Keywords screen will be displayed.
  • Choose the item of metadata you wish to modify and click on it or the edit link on the far right of the screen.
  • The taxonony for the required items of metadata will be shown down the left hand side. Click the arrow to the left of each folder to open a branch of the taxonomy.
  • To change an existing metadata value, click on the folder and it will populate the editing area to the right. The metadata name is the name that appears on the metadata selections made by the user. External reference is an optional identifier for a third party classification scheme (if required). Synonyms are alternative terms for metadata which match on searches but do not get shown to users.
  • To add a metadata, you must pick an existing one that it will become a child of. If this is a top level metadata entry, choose the root term. Then enter the required details for the metadata and click update.
  • To change the position of a metadata term in the taxonomy, use the direction controls (left, right, up, down) to move it or you can drag and drop the metadata term into a different location.
