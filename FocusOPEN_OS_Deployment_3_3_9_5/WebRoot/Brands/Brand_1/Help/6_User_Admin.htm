About User Roles

[top ^ | close]

Having registered with the system, users of FocusOPEN Digital Asset Manager may access the system via either an internal or external account:

Internal User & Administrator Accounts

Employees who have registered internal email addresses, have internal user accounts. Internal accounts support both user and administrator roles. After internal user accounts expire, users receive a security prompt to reactivate their accounts. Only super administrator accounts can be set to never expire. Administrator roles are:

  • Upload Users
    Have access to Asset Management and Order Management features. Upload Users are responsible for all assets uploaded to the system by themselves and are required to respond to usage requests for those assets which have download rights applied. View rights are restricted to those assets or orders for which they are responsible.

  • Brand Administrators
    • Have access to Asset Management, Order Management and some User Management features. Business User Administrators support Upload Users in their Brand and can view all users, assets, orders and reports specific to their Business Unit.

  • Super Administrators
    Have access to all administration features, including Content Management and Utilities. Super Administrators support Upload and Brand Administrators and can view all users, assets, orders and reports system-wide.

Home Use Permitted
Checking the 'Home Use Permitted' checkbox (available from the Edit User screen), allows users to access the Media Library from outside the corporate network (for example, from home). Note: This option is available to employees only.

External User Accounts

Approved third-party companies (for example: agencies) who access the system from outside the corporate network and who have had applied to have their domain registered with the system by a Super Administrator, have external user accounts. External accounts do not support administration roles and automatically expire ten days after activation.

Manage Users

Available to: Super & Brand Admin
[top ^ | close]

The User Management features allows Administrators to find users, edit user accounts and view user statistics.

To Find A User
  • Select User Admin > Manage Users from the main navigation in the Admin Area. The Manage Users screen will be displayed.
  • The Manage Users screen displays all users by default (regardless of account status: expired, suspended or active) applicable to administrator role.
  • Brand Administrators: Only view users in their own Brand. To find a specific user, enter their email into the "User Email" textbox.
  • Super Administrators: View all system users. To find a specific user, enter their email into the "User Email" textbox. Results can be filtered by Brand.
  • Press the Search button to filter the results displayed in the table below. The View Users List table displays the following column headings: User Name, User Email, User Type, Company, Brand, Expired, Suspended, Last Login.
  • To view expired and suspended accounts only, check the show expired and suspended accounts only checkbox beneath the search controls. To sort results by a column heading, click on the link beneath the column title.


    • To Edit User Details

  • Having found a user account to edit, you can edit the details of individual user accounts by either: clicking on the link under the user's name or: clicking the 'Edit' link at the far right of the View User List table. The Edit User screen is displayed.
  • Click on the 'Stats' link at the far right of the View User List table, to view asset statistics
  • On the Edit User screen, revise the information in the fields presented as required. Note: Fields marked with a * are required.

    Password
    Leave the password field blank if you do not want to edit the user's password. Note: You must notify users of any changes made to their password or email details.
    Staff User
    If Staff User is chosen, only internal companies will be displayed

    Last Logged In
    Shows the date on which the user last logged into the system. If a users has not logged into the system, "Never" is displayed.

    Password Expires
    Shows the date on which the user's password will expire. If the password has expired, the date is displayed in red.
    Account Expires
    Shows the date on which the user account will expire. If the account has expired, the date is displayed in red. Only Super Administrators may have accounts which never expire.

    Permissions
    Shows the user's role: Normal, Upload User, Brand Administrator and Super Administrator. The Brand Administrator role is not used on FocusOPEN Digital Asset Manager

    Approver
    Allows the user to delegate who approves any assets where they are involved in workflows.

    Workflow
    If the user is required to have any requests to publish assets checked via an upload workflow, this option should be ticked and a workflow chosen. The user can be offered the option to choose their own workflow if required.

    Account Suspended?
    Will prevent users from accessing the system.

    Account Never Expires
    Allows an account to be set to never expire.
    Password Never Expires
    Prevents a password expiring so the user does not need to change it.

    Home Use Permitted
    Users may access the system from an alternative IP address. Note: This option is available to employees only and is not displayed if IP address restrictions are disabled.

  • Having revised the account details of a user, press the Save button to save the amends. Note: Press the Cancel button to cancel the editing process. Any changes will be lost.
  • You must notify users of any changes made to their accounts.

Manage Companies

Available to: Super & Brand Admin
[top ^ | close]

The Manage Companies feature allows Super Administrators and Business User Administrators to add and edit the details of approved external users (for example, third-party suppliers). External users must first apply for access via the Registration screen. Once submitted, their registration request will be emailed to a Super Administrator for approval.

To Approve An External User:
  • Follow the link sent in the email dispatched by the system after a registration by an external user has been completed. The User Approval screen will be displayed.
  • Review and respond to the request by either approving access (check the approve checkbox) or rejecting access (check the reject checkbox). Note: If you choose the reject the request, add notes in the Your Response field which detail your decision.
  • External users who are approved will have their domain automatically added to the Manage Companies list.

    • To Edit The Account Of An External User:
  • Select User Admin > Manage Companies from the main navigation in the Admin Area. The Manage Companies screen will be displayed.
  • In the View Companies List, edit the name, email domain and status (internal is checked) of a company by clicking the "Edit" link at the far right of the View Companies List table. All fields will be displayed as editable text fields. Click the "Update" link to save any amends made. To sort results by a column heading, click on the link beneath the column title.
  • To add a new company to the View Companies List, enter the company's name and email domain into the textboxes displayed at the end of the list and select the status (internal is checked). Press the "Add" button to add the company to the list. Note: All fields are required.
  • If more than one brand is available on your instance of FocusOPEN, you will be required to choose those brands that users from the new company will have access to.

Manage IP Addresses

Available to: Super Admin Only
[top ^ | close]

The Manage IP Addresses feature allows Super Administrators to add, edit and delete the IP addresses of internal networks (for example, to allow access to subsidiaries). Only internal users are subject to restricted access based on IP address. IP Address restrictions are switched off by default and must be enabled by an engineer (see the deployment guide if you are a systems administrator).

  • Select User Admin > Manage IP Addresses from the main navigation in the Admin Area. The Manage IP Addresses screen will be displayed.
  • In the View IP Addresses List, edit the IP address of a company by clicking the "Edit" link at the far right of the View IP Addresses List table. All fields will be displayed as editable text fields. Click the "Update" link to save any amends made. To sort results by a column heading, click on the link beneath the column title.
  • To add a new IP address to the View IP Addresses List, enter the IP address into the textbox displayed at the end of the list. Press the "Add" button to add the IP address to the list. Note: All fields are required.
  • To delete an IP address from the View IP Addresses List, click on the "Delete" link at the far right of the View IP Addresses List table. Confirm deletion to remove the IP address from the list.

Manage Workflows

Available to: Super Admin Only
[top ^ | close]

The Manage Workflows feature allows Super Administrators to change who is involved in publication (uploading) of assets. You can add as many workflow as required and assign them to users (or let them choose their own).

  • Select User Admin > Manage Workflows from the main navigation in the Admin Area. The Manage Worfklow screen will be displayed.
  • Locate the workflow you wish to amend and click edit.
  • The workflow designer will be shown.
  • Supply a name for the workflow.
  • The list at the top of the screen shows all available users, if you click on a user and then click 'add to workflow' they will be added to those users who approve workflow processes.
  • Use move up, move down to change the approval order and remove to take a user out of the list of approvers.
  • Click 'save' to store the changes to a workflow.
