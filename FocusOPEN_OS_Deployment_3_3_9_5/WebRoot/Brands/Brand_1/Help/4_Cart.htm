Ordering Assets on the System: About The Cart

[top ^ | close]

About the Cart

To download restricted assets from the system, users must first place an order. Orders are made via the Cart feature, which is available from all web catalogue interfaces via the "Cart" link in the main navigation. Once all assets have been added to the Cart, users can place an order with the system. Administrators will be notified of assets requiring usage approval in an order and will respond to each application. Assets which do not require usage approval will be made immediately available for download via My Orders. Note: Unrestricted assets can be downloaded directly from the Search Results interface via the 'download' icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View.

Managing Assets In The Cart

[top ^ | close]

There are two ways to add assets to the Cart: Directly from the search results or from a Lightbox:

Adding/Removing Assets in the Cart from the Search Results

To add an asset to the cart from search results, press the "add to cart" icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View. Once added to the cart, the icon will be highlighted. To remove the asset from the cart, simply click on the highlighted icon.

Adding/Removing Assets in the Cart from Lightboxes

To add assets to the cart from a lightbox (main menu > lightbox), either:

  • Select a lightbox from the "Selected Lightbox" dropdown menu (Note: The default lightbox is displayed by default)
  • Select the asset or assets you want to copy or move via the "Select This Asset" checkbox displayed beneath the asset details
    Note: To select all assets in a lightbox, check the "Select All Assets" checkbox.
  • Select the "Move Or Copy Assets To Cart" link available from the right hand panel (under the "Assets" link)
  • Select whether you wish to move or copy the assets to the cart.
    Note: Moving assets will remove the selected assets from the original lightbox. Copying assets will retained the selected assets in the original lightbox.
  • Press the "OK" button to move or copy the select assets to the cart.
  • Note: Assets can also be added to the Cart from the persistent lightbox.

or:

Press the "add to cart" icon which is displayed in the row of icons under each asset in the Lightbox. Once copied to the cart, the icon will be highlighted. To remove the asset from the cart, simply click on the highlighted icon.

Removing Assets From The Cart

To remove assets in the Cart, select the asset or assets you want to remove via the "Select This Asset" checkbox displayed beneath the asset details and click the "Remove Selected" link displayed at the top and bottom of the asset list or click on the highlighted "remove from cart" icon beneath each asset.

About Download & Usage Restrictions

[top ^ | close]

Not all assets are available for immediate download from the system. Some assets may have usage or copyright restrictions which must be considered by an Administrator. Only if criteria are met, will these assets be approved for use and become available for download via the My Orders.

About Download Restrictions

If an asset has download restrictions applied, a red border is displayed around the asset thumbnail in the search results (Thumbnail View). Rollover the asset thumbnail or click on the information icon beneath the asset thumbnail to view more information on usage and download restrictions. In the cart, the download restrictions are displayed to the right of the asset details. If no approval is required to download an asset, on processing the order this asset will become immediately available to download from My Orders. If approval is required, your application will be sent for review to an Administrator before assets are made available to download. Note: Unrestricted assets can download directly from the Search Results interface via the 'download' icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View.

About Usage Restrictions

Usage restrictions describe any copyright or other considerations for use. Usage restrictions are available to view from the Asset Details or from the details displayed when rolling over the asset thumbnail in the search results (Thumbnail View).

About Intended Use Notes

If an asset added to the Cart has download restrictions applied, you must complete the Intended Use text field, providing details on how you propose to use the asset selected and specify the date required via the Date Required field. Press the "Save" button beneath each asset to save these details. Note: Where applicable, intended use and date required must be specified before an order can be processed. Note: You may be contacted by an Administrator for further information regarding intended use.

Processing Orders

[top ^ | close]

To process an order:

  • Ensure that intended use notes are complete for all applicable assets in the Cart
  • Check the "Terms & Conditions" checkbox to confirm that you will adhere to the Terms & Conditions of Use of the FocusOPEN Digital Asset Manager and that you are authorised to use the system.
  • Press the "Process Cart" button to process the order.
  • Assets in the order which do not require approval are available immediately from My Orders. You will notified by an Administrator when restricted assets have been approved for your use.

Persistent Cart Menu

[top ^ | close]

The persistent cart menu is accessed from the persistent lightbox menu, available at the bottom of the screen when search results are displayed. The persistent cart is a convenient way of managing your order whilst searching for assets and provides the following features:

  • Current View: The "Current View" dropdown menu allows you to toggle between the persistent cart menu and the persistent lightbox menu.
  • Cart Options: These options allow you to proceed to order and remove all assets from the cart.
  • Hide/Show: The "Hide/Show" links allow you to open and close the persistent menu.

To add assets to the persistent cart from search results, press the "add to cart" icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View. Once added to the persistent cart, this icon will be highlighted. To remove the asset from the persistent cart, simply click on the highlighted icon. You can navigate the assets in the persistent cart by clicking on the next and previous arrows at either side of the thumbnails displayed.
