Uploading Assets

Available to: All Admin & Upload Users
[top ^ | close]

The Upload Assets feature allows Upload Users and Administrators to get asset files from their local hard drive or network folder onto the server, so that they can be catalogued and published to FocusOPEN Digital Asset Manager for other users to search for and download.

To upload an asset to the server:

  • Select 'Manage Assets' > 'Upload Assets' from the main navigation in the Admin Area. The Upload Assets screen will be displayed.
  • Select the file to upload by pressing the 'Browse' button alongside the 'Select File' field. (Required field) The 'Choose File' dialogue window is displayed.
  • Navigate your local hard drive or corporate network drive to locate the required file. Press the 'Open' button on the dialogue window to select the file. The file path will be displayed in the 'Select File' field. See Batch Uploads: Uploading Zipped Files for information on how to upload more than one asset at a time.
  • Check the 'email me when previews have been processed' checkbox to receive an email notification once the system has processed the file and an asset preview is available. Emails will be dispatched to your registered inbox.
  • Normally, FocusOPEN will choose the asset type associated with the file extension. You can override this and choose an asset type to assign an asset to from the Asset Type drop down menu.
  • Press the 'Upload' button to start the file upload. A progress bar is displayed which indicates the duration of the upload and the data transferred so far. Note: Please ensure that the copyright and usage rights are available before uploading your assets. All copyright, usage and download rights must be specified before assets can be published to the system.
  • To cancel the upload process, press the 'Cancel' button displayed beneath the upload status bar. Note: Navigating from the Upload Assets screen will also terminate the upload process. A confirmation alert will be displayed should you choose to do this. Press 'OK' to continue (and cancel the current upload) or 'Cancel' to return to the Upload Asset screen without termination of the upload process.

Batch Uploads: Uploading Zipped Files

Available to: All Admin & Upload Users
[top ^ | close]

To batch upload (upload more than one asset at a time):

  • Select 'Manage Assets > Upload Assets' from the main navigation in the Admin Area. The Upload Assets screen will be displayed.
  • Select the zipped file to upload by pressing the 'Browse' button alongside the 'Select File' field. (Required field) The 'Choose File' dialogue window is displayed. Note: ALL files within the zipped file must be the same asset type (e.g. all images, videos, audio or documents). Mixed file formats are supported.
  • Navigate your local hard drive or corporate network drive to locate the required zipped file. Press the 'Open' button on the dialogue window to select the zipped file. The file path will be displayed in the 'Select File' text field.
  • Check the 'email me when previews have been processed' checkbox to receive an email notification once the system has processed the file and an asset preview is available. Emails will be dispatched to your registered inbox.
  • Press the 'Upload' button to start the file upload. A progress bar is displayed which indicates the duration of the upload and the data transferred so far. Note: Please ensure that the copyright and usage rights are available for all assets in the zipped file before uploading your assets. All copyright, usage and download rights must be specified before assets can be published to the system.
  • To cancel the upload process, press the 'Cancel' button displayed beneath the upload status bar. Note: Navigating from the Upload Assets screen will also terminate the upload process. A confirmation alert will be displayed should you choose to do this. Press 'OK' to continue (and cancel the current upload) or 'Cancel' to return to the Upload Asset screen without termination of the upload process.
Note if you wish to upload a zip file and not have FocusOPEN separate the component files contained within the zip, check the not a batch (preserve zip as generic file) option. This can be useful if you need to upload a collection of multiple assets that cannot be broken apart or a very large file which must always be delivered compressed.

About Processing Assets

Available to: All Admin & Upload Users
[top ^ | close]

For the majority of assets uploaded, a preview or proxy of that asset is produced for display on the system. In the case of video, a fast-loading, high-quality preview of the video is produced so users browsing for video content can watch the video before downloading the original file.

To generate a video preview, files must first be processed by the system. As this may take some take some time, a temporary thumbnail is shown. Administrators can opt to receive an email notification when video assets have been processed, by checking the 'email me when previews have been processed' checkbox on the Upload Assets screen.

Cataloguing Assets

Available to: All Admin & Upload Users
[top ^ | close]

The cataloguing feature allows Administrators to catalogue assets which have previously been uploaded to the server. Cataloguing is the process of attributing metadata to an asset so that it can be retrieved in searches and used without breach of copyright or other usage rights.

Step One: Finding & Selecting Assets To Catalogue

To find and select assets which have been uploaded to the system to catalogue:

  • Select 'Manage Assets > Catalogue Assets' from the main navigation in the Admin Area. The Catalogue Assets: Step One screen will be displayed. Note: Only assets uploaded by you will be available for you to catalogue. If you have not uploaded any assets to the system and none are available for you to catalogue, the system will display feedback to this effect.
  • To retrieve uploaded assets to catalogue: Select the type of asset which you want to catalogue (to see all available assets, select the 'All' option from the 'Type' dropdown menu). A list of uploaded assets is displayed in the 'File Reference' list menu below.
  • To select the assets to catalogue in this session: Click on an asset file reference in the 'File Reference' list menu. For your reference a thumbnail of the asset selected is displayed alongside the list menu. Note: Hold down the CTRL key and click to select more than one asset to catalogue or to deselect an asset. To select all assets displayed in the list menu, click the 'select all assets' link beneath the list menu.
  • Press the 'Catalogue Selected' button to confirm your selection for this session and proceed to the next step.

About Templates

Available to: All Admin & Upload Users
[top ^ | close]

Templates provide a quick and convenient method for reapplying the metadata added for one asset to all other assets selected for cataloguing in a session. To use templates, select one of the following options from the Select Templates section:

  • Select the 'use the first asset as a template for the others' radio button to nominate the first asset which you catalogue as a template for all other assets catalogued in that session. In this way, once the first asset has been catalogued, this data will be applied to all subsequent assets which you catalogue in that session. Note: If you need to make changes to any subsequent assets, you will be able to do so on a case-by-case basis.

  • Select the 'use another asset as a template' radio button to nominate an asset which has previously been catalogued, as a template for all others. In this way, the data from a catalogued asset can be used as a template for all subsequent assets which you catalogue in that session.

    To nominate a catalogued asset as a template for all others in a session:
    Enter the reference number of the asset into the text field next to the 'use another asset as a template' radio button. To view the details for the asset entered, click the 'view' link to the right of the text field. The Asset Details window is displayed. Note: If you need to make changes to any subsequent assets, you will be able to do so on a case-by-case basis.

    To find the reference number for an asset which has already been catalogued, select the "Catalogue" link from the Admin main menu and search to find an asset.

  • If you do not want to use templates, select the 'don't use templates' radio button. Note: this option is selected by default.

Step Two: Cataloguing Selected Assets

Available to: All Admin & Upload Users
[top ^ | close]

Having selected asset(s) to catalogue, metadata and other information can then be specified for each asset selected in turn.

To add metadata to a selected asset:

On the Catalogue Asset > Step Two screen, enter information into the fields presented. Note: Fields marked with * are required. A preview of the asset to be catalogued is displayed above the metadata fields for that asset. You can choose to replace this asset with another file or download the original asset. Note: When replacing an asset with another file, you must select a file of the same type as the original asset (i.e. video or image). Files with large filesizes, for example video, may take some time to download in their original format.

Different types of asset present fields relevant to that type. For example, video assets detail information on running time (in hours, minutes and seconds), whereas image assets detail information on orientation (portrait, landscape or square).



About Metadata

Available to: All Admin & Upload Users
[top ^ | close]
Automated Metadata
Where possible, the system will extrapolate information about an asset from the file, for example: File Size, File Format, File Name and File Metadata (EXIF, XMP or IPTC data).

General Metadata
Information about the asset, for example: Originator, Title, Asset Type.

Subject Metadata
This will be unique to each implementation and depends on how your administrator has configured the metadata for your edition of FocusOPEN. You may have a number of drop-down hierarchical menus or hierarchical checkboxes to choose from in this section. All subject oriented metadata options are chosen from a controlled vocabulary.

Categories
The category folders an asset is assigned to. Also required is a Primary Category for an asset, this appears across the top of the asset detail pop-up window.

Descriptive Metadata
Information which describes the content of the asset, for example Short Description, Target Group, General Notes. The Short Description is an important field in which specific details regarding the asset can be entered and should be considered with care. (Max characters: 400)

Copyright, Usage & Download Rights

Available to: All Admin & Upload Users
[top ^ | close]
Copyright Owner
The organisation (or individual) to which the copyright belongs.

Usage Restrictions
Any restriction on the usage of an asset. This may be particular to a context (for example, use in print media) or general usage (for example, logo minimum size and exclusion zone).

Attached Files
You can attach additional files to assets (in addition to the asset itself). Typically this might include licence documents, logo usage guidelines, fonts or other items that need to accompany the asset. Attachments can be restricted using the same download restrictions as the asset itself (if left unchecked, users can download them from the asset detail pop-up).

Linked Assets
Enter the ID number of any assets that are to be linked to this one. When you tab off the filed, the title of the target asset will be shown. Links are bi-directional so a link to the source asset will be added to the target automatically.

Download Restrictions
Download restrictions prevent users from downloading an asset from FocusOPEN Digital Asset Manager without approval. Download restrictions are applied by default and include:

  • Permission required to download
    Requires non-administrator users to request to download an asset. This commences usage approval (download) workflow where administrators decide if a request is approved or rejected.
  • Watermark Preview
    A watermark is superimposed over the asset preview so that the preview cannot be downloaded and reused without permission. Watermarks can be added to both image and video previews.
  • Not visible to external users (e.g. agencies)
    The asset (including the asset preview) is not visible to users outside of the corporate network and is not returned in search results.

Publication (Expiry & Publish Dates)


Available to: All Admin & Upload Users
[top ^ | close]
Information related to the asset's publication status on the system. A publication date must be set for all assets, but no expiry date can also be specified if an expiry is not required. The publication date denotes the date on which the asset will be available for users browse and download via the catalogue. The expiry date denotes the date on which the asset will be removed from searches and will no longer be available to download.

Note: users who have ordered an asset prior to its expiry date will still view a reference to that asset in their order history but will be no longer able to download that asset.

To Publish An Asset:

  • Set the publication date for the date on which you want the asset to become available. Note: The current date will be selected by default.
  • If required, set the expiry date for the date on which you want the asset to become unavailable. If the asset has been acquired from a third party and there is an embargo date beyond which it should not be used, make sure you enter it here. Note: A date two years in advance of the current date will be selected by default. To not specify an expiry date option, set the radio button to Asset Expires: No.
The next section depends on whether you are required to submit assets to a workflow
  • If not, you will have a 'Publish This Asset' checkbox which can be ticked to confirm that the asset can be published to the system. Note: Assets will not be available to users via the catalogue unless this checkbox is checked. Checking the 'publish this asset 'checkbox will immediately publish the asset to the system if the current date is after the publication date set and before the expiry.
  • If you are required to submit assets to a workflow, this checkbox will say 'submit to workflow' instead. When you save the asset, emails will be sent to the administrators to ask them to review your request to publish and either approve or reject it. Once the asset is submitted, it cannot be edited again unless approved or rejected.
  • Having applied metadata to an asset, press one of the following buttons to catalogue the asset. (Note: Asset metadata will not be saved unless one of the Save options below is selected):

    Save & Previous
    Available if you have selected more than one asset to catalogue in this session. Press the 'Save & Previous' button to save the information applied to the asset and move to the previous asset in the selection (reference number is displayed on the button). Note: This option is not available if the asset is the first in the selection.

    Save & Exit
    Press the 'Save & Exit' button to save the information applied to the asset and exit from the cataloguing process. Note: You will have to retrieve and re-select assets to be catalogued if this option is exercised.

    Save & Next
    Available if you have selected more than one asset to catalogue in this session. Press the 'Save & Next' button to save the information applied to the asset and move to the next asset in the selection (reference number is displayed on the button). Note: This option is not available if the asset is the last in the selection.

    Cancel
    Press the 'Cancel' button to cancel the cataloguing process. Note: Any changes will be lost.
Note: it is possible to save an asset without publishing it or submitting it for approval. Also, assets that are only being saved rather than published do not need to be completed in full and required fields can be left blank until you are ready to publish or submit for approval.

Editing Assets: View Asset List

Available to: All Admin & Upload Users
[top ^ | close]

Having uploaded and catalogued an asset, asset information can be edited for all assets available to the system and asset statistics can be viewed.

To Find An Asset To Edit
  • Either: Search for the asset via the catalogue and view the Asset Details pop-up window by clicking the [i] information icon.
  • If the asset was uploaded by yourself (Upload User), belongs to your Brand (Brand Admin) or if you have Super Admin rights, an Edit link will be available under the asset preview.
  • Click on the Edit link. The Edit Asset screen will be presented with the metadata for the selected asset displayed.


  • Or: Select 'Manage Assets > View Asset List' from the main navigation in the Admin Area. The View Asset List screen will be displayed.
  • The View Asset List displays all assets available to you on the system. All assets are displayed by default (regardless of publication status)
  • To search for an asset to edit use the search controls available at the top of the screen. Enter a search term into the 'Search' text field and filter the results by asset type. Further options may be available depending on your user role.
  • Upload Users: Can also filter search results by Category and Publication Status.
  • Brand Administrators: Can filter search results Upload User, Category and Publication Status.
  • Super Administrators: Can filter search results by Brand, Upload User, Category and Publication Status.
  • The Show Thumbnails checkbox allows a thumbnail to be displayed also.
  • Press the 'Search' button to filter the results displayed in the table below. The View Asset List table displays the following column headings: Reference number, Type, Upload User, Title, Date Produced (month and/or year), File Size, Expiry Status (yes or no), Date of Expiry and Publication Status (yes or no). Note: To sort results by a column heading, click on the link beneath the column title.
  • The checkbox allows individual assets to be selected. Selected assets can be renewed (if any expire), delegated to another user, reassigned to an alternative category or deleted. For batch deletions, the user is require to re-enter their password.
  • If an asset is pending workflow review, the publication status will say 'pending'. The original upload user or an administrator may click on this link and approve/reject an asset.

To Edit An Asset

  • Having found an asset to edit, you can edit the metadata of individual assets in the View Asset List by either: clicking on the link under the asset reference number or: clicking the 'Edit' link at the far right of the View Asset List table. The Edit Asset screen is displayed.
  • Click on the 'Stats' link at the far right of the View Asset List table, to view asset statistics
  • On the Edit Asset screen, revise the information in the fields presented as required. Note: Fields marked with a * are required. To find out more about a specific field, roll over the field title to see further information.
  • Having revised the metadata of an asset, press the 'Save' button to save the metadata amends and return to the 'View Asset List'. Note: Press the 'Cancel' button to cancel the editing process. Any changes will be lost.
