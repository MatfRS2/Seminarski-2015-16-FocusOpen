Finding Assets To Use

[top ^ | close]

About Searching

Once logged on, you can search or browse for digital assets from all web catalogue interfaces. The search feature is located to the top left of each screen, below the main navigation. To perform a search:

  1. Enter a search term into the search field. [more information on search terms]
  2. Select the type of assets to search for from the Media menu.
    To search for all types of media, select the "All" option.
  3. Press the "Go" button to start the search.

The system will display all results found that match your search criteria. If no results are found, the system will return "No Results Found". Revise your search and try again.

Search Tips

[top ^ | close]

About Search Grammar

To search for assets in the FocusOPEN Digital Asset Manager you must enter a search term (or terms) into the search field. You can use search grammar to further refine your results. Note: Case is ignored when searching for terms. Follow the tips below to see how you can get better results from your searches:

  1. Tip 1: Using more than one word
  2. Tip 2: Using speech marks to find exact phrases ("")
  3. Tip 3: Using OR
  4. Tip 4: Using NOT (-)
  5. Tip 5: Using Wildcards (*)

Using more than one word

If you add more than one search term, then the system will look for assets which relate to all those terms. For example:



Will return all results to which the words 'Bunch' AND 'Grapes' have been applied. Note: You do not have to type the word 'AND' between search terms. The system will look for assets which relate to all search terms entered with spaces in between. This will narrow your search criteria and is useful if too many results are being found. This technique can also be used to search within results.

Using speech marks to find exact phrases ("")

Use speech marks around a search phrase to find assets to which that exact phrase has been applied. For example:



Will return all results to which the exact phrase "Bunch of Grapes" has been applied. This will narrow your search criteria and is useful if too many results are being found.

Using OR

Use 'OR' between search terms to find assets to which any of the search terms have been applied. For example:



Will return all results to which either the word 'Bunch' or 'Grapes' has been applied. Note: This includes assets to which both words have been applied. This will widen your search criteria and is useful if no search results are being found.

Using NOT (-)

Use NOT (represented by - ) to find assets which apply to the term(s) to the left of the '-', but which do not apply to the term(s) to the right of the '-'. For example:



Will return all results to which the word 'Bunch' applies, but not the word 'Grapes'. This will narrow your search criteria and is useful if too many results are being found. Note: There should be no space between the proceeding search term and the '-'.

Using Wildcards (*)

Use wildcards (represented by *) to find assets to which the term or any related nouns, adjectives, verbs and adverbs have been applied. For example:



Will return all results to which the term 'Bunch' or any related words 'Bunched', 'Bunching', 'Bunches' have been applied. This will widen your search criteria and is useful if no search results are being found. Note: There should be no space between the search term and the '*'.

Search Filters

[top ^ | close]

The Search Filters feature is designed to assist you in refining your search for better results. To access the Search Filters interface, click on the Show Filters [+] link located on the top right of all web catalogue interfaces. The Search Filters interface will be revealed immediately below this link. Use the dropdown menus displayed to filter your search by common criteria, such as file size or date produced. Select 'All' from lists to include all options within that criteria set. Having made your selection, press the "Search" button to start your search. Note: Where more than one option is selected, the system will search for assets which apply to either option selected. This will widen the search criteria and is useful if too few results are being found. Click on the Hide Filters [-] link to close the Search Filters interface.

Search Categories

[top ^ | close]

The Search Categories feature is designed to enable you to browse for assets via a directory folder structure, similiar to your local or network drive. To access the Search Categories interface, click on the Show Categories [+] link located on the top right of all web catalogue interfaces. The Search Categories interface will be revealed immediately below this link. Use the breadcrumb menu displayed to open categories (folders) and navigate the directory structure to locate digital assets. Assets within each category (folder) are displayed in the search results below. Click on the Hide Categories [-] link to close the Search Categories interface.

Search Within Category

You can combine search and browse features with the Search Within Category feature, located to the bottom of the Search Categories interface. Check the checkbox to search for terms that relate to digital assets within the current open category only. This will narrow the search criteria and is useful if too many results are being found.

Search Results

[top ^ | close]

Thumbnail & List View

Search results can be viewed in two different formats, which can be selected by clicking either the "View As.." link or the icon in the Current View display. Note: The number of assets viewed per page can be selected on both view formats via the "Show" menu to the right of the Current View display. Use the paging links beneath the "Show" menu to navigate your search results.

Thumbnail View
Shown by default, thumbnail view shows a preview of each asset returned in the search results on a grid. This view is useful for judging the visual content of an asset.

List View
List view shows a text description of each asset returned in the search results in a list. This view is useful when many assets are required to be viewed on screen at once.

View Asset Details

[top ^ | close]

The View Asset Details interface provides a fast-loading preview of the asset and further information about the asset, such as; date produced, file size and copyright owner. To view asset details from search results:

  1. In Thumbnail View: Click on the information icon [i] displayed beneath the asset thumbnail. The View Asset Details pop-up is displayed.
  2. In List View: Click on the information icon [i] displayed to the far right of list. The View Asset Details pop-up is displayed.
  3. Click on the "Close Window" link at the bottom of the The View Asset Details pop-up to close the window.
