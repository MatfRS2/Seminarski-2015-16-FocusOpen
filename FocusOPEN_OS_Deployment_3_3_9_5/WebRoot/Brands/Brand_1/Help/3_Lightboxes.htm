Organising Assets on the System

[top ^ | close]

About Lightboxes

Lightboxes provide a facility for you to organise assets before making an order. Lightboxes can be added, renamed, removed, duplicated, merged and shared with other system users. Once logged on, Lightboxes can be accessed from all web catalogue interfaces via the "Lightbox" link in the main navigation. Although most lightboxes are private, some lightboxes have public status. These public lightboxes are created by Administrators to highlight asset collections to users and are non-editable by users. Public lightboxes are available to view via the Selected Lightbox dropdown menu (displayed with 'public' in brackets after the title).

Adding/Removing Assets in Lightboxes from the Search Results

To add assets to the default lightbox from search results, press the "add to lightbox" icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View. Once copied to the lightbox, the icon will be highlighted. To remove the asset from the default lightbox, simply click on the highlighted icon.

Managing Assets in Lightboxes

[top ^ | close]

To manage assets in your lightboxes, click on the Lightbox link which is available from the main navigation on all web catalogue interfaces. Assets in lightboxes can be managed in the right hand panel under the Assets link (shown by default). Assets can be managed in lightboxes in the following ways:

Removing Assets From a Lightbox

To remove an asset or assets from a lightbox:

  • Select a lightbox from the "Selected Lightbox" dropdown menu (Note: The default lightbox is displayed by default)
  • Select the asset or assets you want to remove via the "Select This Asset" checkbox displayed beneath the asset details
    Note: To select all assets in a lightbox, check the "Select All Assets " checkbox.
  • Select the "Remove Assets From This Lightbox" link available from the right hand panel (under the "Assets" link)
  • Press the "Remove" button to remove the select assets from the lightbox.
    Shortcut: Selected assets can also be removed by clicking the "Remove Selected" link at the top and bottom of the asset list.

Copying/Moving Assets To Another Lightbox

To copy or move an asset or assets to another lightbox:

  • Select a lightbox from the "Selected Lightbox" dropdown menu (Note: The default lightbox is displayed by default)
  • Select the asset or assets you want to copy or move via the "Select This Asset" checkbox displayed beneath the asset details
    Note: To select all assets in a lightbox, check the "Select All Assets" checkbox.
  • Select the "Move Or Copy Assets To Another Lightbox" link available from the right hand panel (under the "Assets" link)
  • Select the lightbox to move or copy the assets to from the dropdown menu
  • Select whether you wish to move or copy the assets to the selected lightbox.
    Note: Moving assets will remove the selected lightbox from the original lightbox. Copying assets will retained the selected assets in the original lightbox.
  • Press the "OK" button to move or copy the select assets to another lightbox.

Copying/Moving Assets To The Cart

To copy or move an asset or assets to the cart:

  • Select a lightbox from the "Selected Lightbox" dropdown menu (Note: The default lightbox is displayed by default)
  • Select the asset or assets you want to copy or move via the "Select This Asset" checkbox displayed beneath the asset details
    Note: To select all assets in a lightbox, check the "Select All Assets" checkbox.
  • Select the "Move Or Copy Assets To Cart" link available from the right hand panel (under the "Assets" link)
  • Select whether you wish to move or copy the assets to the cart.
    Note: Moving assets will remove the selected lightbox from the original lightbox. Copying assets will retained the selected assets in the original lightbox.
  • Press the "OK" button to move or copy the select assets to the cart.

Managing Lightboxes

[top ^ | close]

To manage your lightboxes, click on the Lightbox link which is available from the main navigation on all web catalogue interfaces. Lightboxes can be managed in the right hand panel under the Lightboxes link. Lightboxes can be managed in the following ways:

  • New Lightbox
  • Rename Lightbox
  • Remove Lightbox
  • Duplicate Lightbox
  • Merge Lightboxes
  • Send Lightbox
  • Set Default Lightbox

New Lightbox

To add a new a lightbox:

  • Select the "New Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Enter a name for the new lightbox (max 25 characters). Note: To make this new lightbox your default lightbox, check the "make default lightbox" checkbox.
  • Press the "Save" button, to add the new lightbox. The lightbox will now be available from the "Selected Lightbox" dropdown menu.

Rename Lightbox

To rename a lightbox:

  • Select the "Rename Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Select a lightbox to rename from the dropdown menu
  • Enter a new name for the selected lightbox (max 25 characters).
  • Press the "Rename" button to rename the selected lightbox.

Remove Lightbox

To remove a lightbox:

  • Select the "Remove Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Select a lightbox to remove from the dropdown menu
  • Press the "Remove" button to remove the selected lightbox.

Duplicate Lightbox

To duplicate a lightbox:

  • Select the "Duplicate Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Select a source lightbox to duplicate from the dropdown menu
  • Enter a name for the duplicate lightbox (max 25 characters)
  • Press the "Duplicate" button to duplicate the selected lightbox. The duplicated lightbox will now be available from the "Selected Lightbox" dropdown menu.

Merge Lightboxes

To merge a lightbox with another Lightbox:

  • Select the "Merge Lightboxes" link available from the right hand panel (under the "Lightboxes" link)
  • Select a source lightbox from the dropdown menu. Note: Assets from the source lightbox will be copied to the target lightbox.
  • Select a target lightbox from the dropdown menu.
  • Check the "Remove Source Lightbox" checkbox to remove the source lightbox once the its assets have been copied to the target lightbox.
    Note: The source lightbox will not be removed if this checkbox isn't checked.
  • Press the "Merge" button to merge the source lightbox with the target lightbox.

Send Lightbox

To send (or share) a lightbox with another system user:

  • Select the "Send Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Select a lightbox to send from the dropdown menu (Required)
  • Enter a subject into the "Subject" text field (Required)
  • Enter a message into the "Message" text field
  • Enter the email address of the user you want to send the lightbox to into the "Recipient Email" text field (Required)
    Shortcut: You can select the email address of a user to whom you have sent a lightbox to before from the dropdown menu below.
  • Enter the email address of a user you want to CC' the lightbox to into the "CC Email" text field
    Shortcut: You can select the email address of a user to whom you have sent a lightbox to before from the dropdown menu below.
  • Set an expiry date on which the lightbox will no longer be available to the recipient via the "Expiry Date" date picker.
  • Press the "Send" button to send the lightbox to the recipient.

Set Default Lightbox

The default lightbox is your primary lightbox and is the lightbox to which assets are added or removed from the search results. When you first access the FocusOPEN Digital Asset Manager, a default lightbox entitled "My Assets" is automatically created for you. However, to nominate another lightbox as the default lightbox:

  • Select the "Set Default Lightbox" link available from the right hand panel (under the "Lightboxes" link)
  • Select the lightbox you want to make the default from the dropdown menu
  • Press the "Make Default" button to set the selected lightbox as the default
    Note: Public lightboxes are read-only and cannot be set as your default lightbox

About Asset Notes

[top ^ | close]

Asset notes allow you to make notes regarding each asset in your lightbox. Notes are available to you and any user you share the lightbox with. To make notes on an asset:

  • Enter your notes for an asset in the "Notes" text field, displayed below the asset details
  • Press the "Save Notes" button to save the notes for each asset

About Lightbox Details (Title, Summary, Status & Notes)

[top ^ | close]

Lightbox details (title, summary, status and notes) provide information on a lightbox. These details are available to all users with whom you share the lightbox and provide a synopsis of its content. By default the lightbox notes are hidden. Click the "Show Notes" link on the bottom right corner of the Lightbox Details to view the notes. To edit the details of selected lightbox:

  • Click the "Edit" link in the bottom right corner of the Lightbox Details panel.
  • Edit the lightbox title in the "Title" text field (max 25 characters)
  • Edit the lightbox summary in the "Summary" text field (max 120 characters)
  • Edit the lightbox notes in the "Notes" text field
  • Press the "Save Lightbox Notes" button
    Note: Lightbox status determines who can view the lightbox. Lightboxes with 'public' status can be viewed by all system users. Lightboxes with 'private' status can be viewed by only yourself and anyone with whom you share the lightbox. The edit status feature is only available to Super Administrators.

Persistent Lightbox Menu

[top ^ | close]

By default, the persistent lightbox menu shows the contents of your default lightbox and is always available at the bottom of the screen when search results are displayed. The persistent lightbox is a convenient way of managing selected assets whilst searching and provides the following features:

  • Current View: The "Current View" dropdown menu allows you to toggle between the persistent lightbox menu and the persistent cart menu.
  • Lightbox Options: These options allow you to add a new lightbox, send (share) a lightbox, view the lightbox (full screen) and add all assets in the lightbox to the cart.
  • Selected Lightbox: The "Selected Lightbox" dropdown allows you to view another lightbox on the persistent menu.
  • Hide/Show: The "Hide/Show" links allow you to open and close the persistent menu.

To add assets to the persistent lightbox from search results, press the "add to lightbox" icon which is displayed in the row of icons under each asset in Thumbnail View, or is displayed in the far right hand column in List View. Once added to the persistent lightbox, this icon will be highlighted. To remove the asset from the persistent lightbox, simply click on the highlighted icon. You can navigate the assets in the persistent lightbox by clicking on the next and previous arrows at either side of the thumbnails displayed.
